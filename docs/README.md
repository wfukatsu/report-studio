# Report Studio ドキュメント

Report Studio（Report Design Studio V2）は、ドラッグ&ドロップで帳票・フォームを設計し、PDF/PNG エクスポートや ScalarDB との連携を行う Web アプリケーションです。日本のビジネス帳票（見積書・請求書・税務様式）に必要な印鑑欄・和暦・縦書き・ふりがなをネイティブサポートします。

このディレクトリは公式ドキュメントの入口です。目的に応じて以下から読み進めてください。

## 30 秒で全体像

```
ブラウザ（React SPA / デザイナー）
   │  {{fieldKey}} でデータを差し込む帳票テンプレートを設計
   │  REST API (/api/v2/*, /api/v1/*)
   ▼
Java / Javalin バックエンド
   ├─ ExpressionEngine（JEXL サンドボックス：計算・条件・検証）
   ├─ PdfRenderer（Apache PDFBox：和文フォント埋め込み・ページ分割）
   ├─ ジョブ基盤（非同期・バッチ PDF、ZIP 出力）
   └─ ScalarDB 3.14（テンプレート・バージョン・回答・テナント・商品マスター）
        └─ SQLite（開発） / 任意の JDBC（本番）
```

- **設計時（デザイナー）**: `{{fieldKey}}` トークンやデータフィールドで「どこに何を出すか」を配置します。
- **プレビュー/出力時**: 同じテンプレートがサンプルデータや ScalarDB の実データに解決され、PDF/PNG/Excel/CSV として出力されます。
- **1 つのテンプレートから複数の出力**: 出力バリアント（宛先ごとの項目マスキング）で、同一テンプレートから宛先別 PDF を作れます。

## まず動かす

最短で試すには [導入方法](./setup.md) の「Docker でのクイックスタート」を参照してください。Node.js や JDK を用意せずワンコマンドで起動できます。

```bash
docker compose up --build
# → http://localhost:8080 を開き admin / changeme でログイン
```

ローカル開発環境（フロント + バックエンド）を立てる場合は [導入方法 › ローカルセットアップ](./setup.md#ローカルセットアップ) を参照してください。
