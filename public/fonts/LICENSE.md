# Fonts license

The `.woff2` files in this directory are subsets generated (fontTools
`pyftsubset`, core + kanji unicode-range slices) from the same Noto Sans JP /
Noto Serif JP files bundled with the server PDF engine at
`server/src/main/resources/fonts/` — see the license note there.
